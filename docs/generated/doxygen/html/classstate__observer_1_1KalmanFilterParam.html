Detailed Description

Parameters for the Kalman Filter observer.

The KalmanFilterParam class extends StateObserverParam to include the process noise covariance matrix Q, measurement noise covariance matrix R, and the initial error covariance matrix P0 specific to the Kalman Filter observer.
